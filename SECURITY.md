# Security Policy

## Supported Versions

The following versions of **WebAI** are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

We take the security of this project seriously. If you believe you have found a security vulnerability, please report it privately.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them via the following method:

1.  **Email**: Send a detailed report to **[Your Email/Contact]**.
2.  **Details**: Please include a description of the vulnerability, steps to reproduce, and potential impact.

You should receive a response within 48 hours. If the vulnerability is confirmed, we will coordinate a fix and a public disclosure timeline.

## Project Integrity

This project utilizes **Docker** to ensure isolated execution environments for the AI models. We recommend always running the project within the provided Docker containers to minimize local system exposure.
